# RST61 — 3D Configurator

## Проект
Сайт-конфигуратор боевой экипировки RST61 (рашгарды) для бренда из Ростова-на-Дону.
Путь на диске: C:\RST61
Репозиторий: https://github.com/squizzyk/rst61-app.git
Живая версия: https://squizzyk.github.io/rst61-app/

## Стек
- HTML5, CSS3, Vanilla JS (ES6+)
- Three.js v0.160.0 (CDN Import Map)
- GLTFLoader + DRACOLoader
- iro.js v5 (скрыто под кнопкой «СВОЙ ЦВЕТ»)
- Архитектура: SPA (4 вкладки)
- Локальный сервер: Live Server на порту 5500

## Git
- Ветка: main (только)
- Push: git push origin main
- НИКОГДА: ветки, PR, worktrees, gh CLI
- Git user: squizzyk, email: nsc90210@gmail.com

## 3D-модель
- Файл: models/rashguard.glb (247 КБ, Draco, 36K вершин)
- Источник: Sketchfab (baselayer_shirt_men.glb), обработана в Blender
- 22 объекта, 3 зоны материалов: FABRIC_1_FRONT (торс перед), FABRIC_3_FRONT (торс остальное), FABRIC_2_FRONT (рукава)
- Ориентация: rotation.x = -Math.PI / 2, rotation.z = 0

## Производство

### Ткань
- Матовый бифлекс 230 г/м², 87% PE + 13% эластан, белый (под сублимацию)
- Расход: 1.3-1.5 м (M-L), 1.5-1.7 м (XL-XXL)
- Ширина рулона 150 см

### Цех (Ростов)
- Работают по «полотном выгоняем, лазером режем» (сублимация + лазерный крой)
- Пошив тестового: 2200₽. Серия от 10 шт — цель 800-1200₽/шт
- Оверлок 5-ниточный, flatlock купят при постоянных заказах

### Себестоимость (серия от 10 шт)
- Ткань 400-600₽ + печать 350-500₽ + пошив 800-1200₽ + фурнитура 100-200₽
- Итого: 1650-2500₽. Цена 4500-5500₽. Маржа 55-70%

## Claude Code workflow
1. VS Code в C:\RST61
2. Терминал (Ctrl+~), команда claude
3. /clear, затем /plan (или Alt+M)
4. Вставить промпт, подтвердить план
5. После пуша: git pull origin main
6. Ctrl+Shift+R в браузере

## Частые проблемы
- Изменения не видны: git pull + Ctrl+Shift+R в инкогнито
- Кривая модель после Blender: проверить rotation.x = -Math.PI/2 и UV
- Зум скачет на масштабе < 100%: решено кастомным wheel-handler
